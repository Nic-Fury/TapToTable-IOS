# 📱 Automatisches Haushaltsbuch mit Apple Shortcuts & Numbers

Willkommen! Dieses Repository enthält alles, was du brauchst, um deine Ausgaben auf dem iPhone in Sekundenschnelle per "Doppeltipp auf die Rückseite" in eine Apple Numbers Tabelle einzutragen.

## 🌟 Features
* **Extrem schnell:** Keine App öffnen – einfach auf die iPhone-Rückseite tippen, Betrag eingeben, fertig.
* **Vollautomatisch:** Der Kurzbefehl (Shortcut) trägt Datum, Kategorie und Betrag selbstständig ein.
* **Übersichtlich:** Die mitgelieferte Numbers-Tabelle summiert alle Ausgaben und Einnahmen automatisch nach Monaten auf.
* **100% Datenschutz:** Alle Daten bleiben lokal in deiner iCloud. Keine Drittanbieter-Apps, keine Abos.

## 📥 Installation

### 1. Die Numbers-Tabelle einrichten
1. Lade dir die Datei `Finanzen-TabToTable Vorlage.numbers` aus diesem Repository herunter.
2. Öffne die Datei in der **Numbers App** auf deinem iPhone, iPad oder Mac.
3. Speichere die Datei in deinem iCloud Drive (z.B. im Ordner *Numbers*). oder wo du willst

### 2. Den iOS Kurzbefehl installieren
1. Lade die `Eintrag zu Finanzen hinzufügen - TabToTable.shortcut`-Datei aus diesem Repository herunter und öffne sie auf dem iPhone.
2. Erlaube dem Kurzbefehl bei der ersten Ausführung den Zugriff auf deine Numbers-Datei.
3. Wähle im Kurzbefehl-Setup die eben gespeicherte Tabelle aus.
<img width="1320" height="461" alt="image" src="https://github.com/user-attachments/assets/cee973fb-e1b6-4d74-950d-5b9bc5552d11" />


### 3. Den "Doppeltipp" aktivieren (Back Tap)
1. Gehe auf deinem iPhone zu **Einstellungen > Bedienungshilfen > Tippen > Auf Rückseite tippen**.
2. Wähle **Doppeltippen** und scrolle ganz nach unten.
3. Wähle unter "Kurzbefehle" den Haushaltsbuch-Shortcut aus.

## 🛠 Nutzung
Immer wenn du an der Kasse stehst oder eine Ausgabe hast:
1. Tippe zweimal auf die Rückseite deines iPhones.
2. Wähle im Menü die Kategorie (z.B. 🍽️ Essen, 🚙 Auto).
3. Gib den Betrag ein.
4. *Erledigt!* Die Numbers-Tabelle im Hintergrund wurde aktualisiert.
